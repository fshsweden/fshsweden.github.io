Introduktion - Den långa versionen!

Jag heter Peter Andersson och bor i Mackmyra, Valbo med min familj om sambo och 3 barn (Tjej 16 bor hemma, Kille 21 studerar i Uppsala (datavetetenskaplig linje), Kille 27 har flyttat hemifrån och efter att ha arbetat ett par år nu tagit upp studierna igen och studerar ekonomi på Högskolan i Gävle).

Jag har ett stort intresse för mitt yrke och spenderar en stor del av min fritid att lära mig nya saker. Det är få miljöer/språk jag inte hunnit titta på. Mina personliga favoriter är Java,Ruby,C/C++ och Python på OSX eller Linux, men jag kan även jobba med C#/.Net på Windows. I Webbmiljö föredrar jag Ruby on Rails som backend, med JS/Typescript på klienten. Gillar även Java Spring Boot med JavaFX på frontend-sidan. Har blivit hyfsat bra på React. Det har blivit mycket Python på sistone, mest pga att det finns så mycket AI/ML i den miljön. Bl.a har jag jobbat en del med OpenCV för tolkning av QR-koder, samt ansikts- och objekt- igenkänning. Väldigt intressant och skulle gärna fortsätta med det. Men framförallt är det Java som intresserar mig.

I och med att mellan-sonen studerar i Uppsala har jag även följt med i hans studier och bl.a plockat upp lite kunskaper i Haskell och MIPS-assembler. Mest för att det är roligt och för att stödja sonen lite i studierna.

Operativsystem

Jag jobbade exklusivt med MS-DOS och sedan Windows från första versionen upp genom versionerna till och med Windows Vista. Sedan c:a 2003 har jag dock mer och mer gått över till Linux, och använder numera helst detta, då jag anser att det överglänser Windows på alla sätt, utom möjligtvis för dataspelande. Senare versioner av Windows har ju ett Linux subsystem, men inte tillräckligt för att nå ända fram.

Java

Jag har jobbat med Java i drygt 10 år, och anser mig vara ganska kunnig. Allt från kommunikation till databaser och web. Även Spring Boot. På senare tid även JavaFX. Har byggt en avancerad trading-applikation med Java 11 och JavaFX. Frontend helt själv och backend i samarbete med kollega.

Javascript med React

Jag har lärt mig React från 0 och byggde på 3 månader en avancerad tradingapplikation i React mot en Java backend. Applikationer använder Redux samt websockets för kommunikation. Dessutom en avancerad "trading grid" med grafiska finesser. Applikationen innehåller även en realtidsgraf med priser och volymer. Har påbörjat en JavaFX-version av samma applikation.

Web

Inom Web har jag kobbat mest med Ruby on Rails som backend, och har provat på de flesta Javascript-ramverk. JavaEE i mindre omfattning samt även lite ASP.Net/C#. Jag har även gjort en mindre applikation i Java Spring Boot, med Thymeleaf, mot en Spring Data databas (MySQL).Min inställning är att de flesta ramverk är ganska lika, och om man kan TCP/IP och HTTP, så är ramverken i sig enkla att sätta sig in i.

Det senaste året har jag dessutom hunnit sätta mig in i Wordpress, och blivit väl bekant med detta verktyg.

Jag är mest komfortabel med MySQL, men har använt andra relationsdatabaser som MS SQL Server och Oracle. Dessutom filbaserade verktyg som Mongo och SQLite samt in-memory-databaser som H2,HSQLDB oc Redis.

Specialkunskaper

Jag har jobbat i finansbranschen eller dess närhet i hela mitt vuxna liv, och har stor kunskap om aktie- och derivathandel (optioner och futures). Jag har gjort handels-system mot de flesta av världens börser. Har stor kunskap om Teknisk Analys.

Övrigt

Jag är speciellt intresserad av Linux och Open Source, och har naturligtvis byggt ett eget rack där jag har mina servrar. Jag gillar att köpa begagnat på Ebay och skruva ihop dem själv.
